# JocdieCo | E-commerce Prototype
A functional e-commerce web application developed as a personal project to showcase full-stack development skills and user-centric design.

## Project Purpose
This is a **concept project** created to demonstrate my ability to build a seamless shopping experience from scratch. It simulates a real-world online store, focusing on product presentation and responsive UI.

## Tech Stack
* **Frontend:** HTML5, CSS3/Sass.
* **Backend:** PHP.
* **Database:** SQL.
